# 🔐 Password Guessing Game (Text-Based)

A simple Python-based terminal game where the user guesses a randomly selected password. The user selects a difficulty level, and the game provides hints for incorrect letters after each guess.

### 🎮 Features

- Difficulty levels: Easy, Medium, Hard
- Random word selection based on difficulty
- Hint system showing incorrect letters
- Tracks number of attempts
- Friendly terminal interface

### 🧠 Concepts Covered

- `random.choice()`
- Loops and conditionals
- String comparison and manipulation
- User input and feedback

1. Run the script:
   ```bash
   python password_game.py
